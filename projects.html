Build A Structure- March-May 2012- Destination Imagination

The objective of this project was to build a structure as a group of seven, using only four materials from a list of options, that would hold up weght while being it is being hit. There are no pictures of the completed structure, but it was a three-pointed pyramid of bamboo shoots, held together at the joints by duct tape and strung through with copper wire. I came up with the copper wire idea and helped string it through holes in the bamboo. I also helped helped with the measuring and cutting of the bamboo. From this I learned how to use tools to measure and modify things, as well as the basics of making a stable structure from this.

Scratch Game Project- August-September 2017 -Computer Sciences and Software Engineering -

The objective of this proeject was to create a multi-level game in Scratch, whose ending and progression could be affected by the player's actions. The project was only two people, my partner and I. The game was a simple maze game with multiple levels, looping sounds, secret levels,and hint and reset functions It had multiple backgrounds as well. I did all of the sound design, creating both functional looping music and sound effects for restarting or completing a level. Not only that, but I came up with the mechanics of the secret levels, programmed the reset button, made the reset button sprite, and did almost all of the project's documentation.

AppInventor Project- August-September 2017 -Computer Sciences and Software Engineering -

The objective of this project was to create an app in MIT's App Inventor with a partner that would take an established base idea and build upon it. We made what was called "Paintpot with Camera. It was a drawing program that allowed you to take a picture and use it as the background to draw on. We expanded upon it by adding multiple ways of erasing what had been draw, RGB color sliders used to make custom colors, the ability to change the size of lines and dots, and the ability to see what custom color has been created by the RGB sliders. I built the base code of the program, along with the UI. I did most of the documentation as well. From this I learned how to design a UI, along with how to package the code for apps.

Scratch Animation Project- August 2017 -Computer Sciences and Software Engineering -

The objective of this project was to take a pre-existing animation in Scratch, and add an additional animation to it. It was a solo project and I decided to amplify the ridiculousness of the animation I was to build on. Where the previous animation had an innocent child talking to an octopus, I expanded upon this, making him slowly realize he was actually schizophrenic. From this, I learned how to create new animations and sprites within scratch, how to make a sprite spin rapidly, and how to apply create music loops. The sprite spinning was the greatest challenge, as when the animation was reset, the sprite did not reset to its inital orientation, and I was forced to figure out how to fix unitended bugs. Most of my animation and musical experience came from this project.
